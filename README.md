# Medical Q&A Chatbot

A publicly accessible medical Q&A chatbot built with a fine-tuned open-source language model and deployed on GCP.

## Goals
- Fine-tune a model on medical question-answer data
- Build a Gradio-based web app
- Deploy it on Google Cloud Run
- Add safety disclaimers and basic guardrails

## Planned Stack
- Python
- Hugging Face Transformers
- PEFT / LoRA
- Gradio
- Docker
- Google Cloud Run# Medical-Q-A-Chatbot
